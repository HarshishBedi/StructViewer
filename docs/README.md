# structviewer Documentation

This folder contains the planning documents for the structviewer web application.

## Structure

- `docs/idea/product-idea.md`: Product vision, goals, and feature scope.
- `docs/design/ui-ux-design.md`: Visual direction, interactions, animation, and UX rules.
- `docs/tech-stack/tech-stack.md`: Recommended modern stack, architecture, and folder strategy.
- `docs/architecture/file-tree.md`: Canonical project tree and file placement conventions.
